# Aliados Fantasma — Demo pública v0.1

Esta es una versión de demostración de la plataforma **Aliados Fantasma**.

## Funciones disponibles

- Inicio de sesión mediante Supabase Auth.
- Dashboard con datos reales.
- Gestión básica de negocios.
- Alta, edición, activación y desactivación de perfiles.
- Perfil público por `slug`.
- Módulos preliminares de solicitudes y promociones.

## En desarrollo

- Flujo completo de aprobación de cambios.
- Panel para propietarios.
- Redes sociales y horarios desde el panel.
- QR, visitas y estadísticas.

## Configuración

Antes de publicar, abre `config.js` y coloca la misma **Project URL** y **Publishable Key** que ya funcionan en tu proyecto local.

Nunca coloques una Secret Key ni la clave `service_role` en el navegador.

Todos los archivos están en la raíz para subirlos directamente a GitHub.
